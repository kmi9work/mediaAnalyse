Если там не VPS, то Ruby on Rails там нету.

Если VPS, то нужно:
Доступ SSH на сервер (желательно с возможностью использовать sudo для установки всех средств программирования)
Логин и пароль к СУБД. Её тип не имеет значения.

Если невозможно дать доступ sudo, то нужны установленные и настроенные:

- SSH
- Консольный редактор (vi / emacs / nano / ...)
- Git
- nginx (или другой веб-сервер)
- Ruby Version Manager (RVM )
- Ruby on Rails
- Unicorn ( HTTP server for Ruby )
- Capistrano
- СУБД
